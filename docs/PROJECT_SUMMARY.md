# Proje Özeti

Bu proje, öğrenci belge/talep süreçlerini n8n ile otomatikleştirmek için hazırlanmış bir örnek workflow setidir. Amaç, manuel talep takibi yerine webhook tabanlı başvuru alma, token tabanlı onay/red akışı, Google Sheets loglama ve Google Docs belge üretimi sağlamaktır.

## Kullanılan teknolojiler

- n8n
- Google Sheets
- Google Docs
- Google Drive
- Google Gemini, opsiyonel mesaj üretimi için

## Temel kazanımlar

- Low-code/no-code otomasyon tasarımı
- Webhook ile entegrasyon
- Token tabanlı işlem kontrolü
- Google Workspace servisleri ile otomasyon
- İş süreci takibi ve loglama

# Sanitization Report

Bu proje GitHub paylaşımı için temizlendi.

Kaldırılan veya örnek değerle değiştirilen alanlar:

- n8n workflow ID
- n8n version ID
- n8n instance ID
- webhook ID
- credential ID ve credential name alanları
- gerçek API anahtarları
- gerçek Google Sheet ID ve cached URL alanları
- gerçek telefon numaraları
- kişiye/kuruma özel demo isimleri

Workflow varsayılan olarak `active: false` durumundadır.
